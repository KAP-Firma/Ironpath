# IRONPATH — Gym Tracker

Ein vollständiger Gym Progress Tracker als Progressive Web App (PWA).

## Features
- 🏋️ Workout Builder mit Übungen, Sätzen, Wiederholungen, Gewicht
- 📊 Interaktives Körpermodell mit Muskel-Rang-System
- ⚡ Quest-System (täglich + wöchentlich) mit XP
- 🏆 1000 Level + Rang-System (E → D → C → B → A → S → S⁺)
- 🔥 Streak-System mit Straf-XP bei Inaktivität
- 👤 Multi-Account System (Gmail-basiert, lokal gespeichert)
- 📱 Installierbar als Android/iOS App (PWA)

## Installation auf Android
1. Öffne die App in **Chrome** auf deinem Handy
2. Tippe auf **⋮ Menü** → **"Zum Startbildschirm hinzufügen"**
3. Die App erscheint mit eigenem Icon auf dem Homescreen
